# Python Chat Application

## Description
This is a simple client-server chat application using Python sockets and Tkinter GUI.

## Features
- Real-time messaging using threading
- GUI-based chat interface
- Graceful exit using 'bye' command
- Error handling for connection issues

## How to Run

1. Run server.py first:
   python server.py

2. Run client.py:
   python client.py

## Usage
- Type message and click Send
- Type 'bye' to exit chat

## Technologies Used
- Python
- Socket Programming
- Tkinter GUI
- Threading
